# Лабораторная работа №1: Типы данных, память и выражения в C

## Цель работы
Научиться работать с базовыми типами данных языка С, понимать связь переменных и памяти, использовать операции и выражения, выполнять ввод и вывод данных, а также понимать переполнение и точность вычислений.

## Используемые технологии
- Язык программирования: C
- Компилятор: gcc
- Стандарт: C11
