# Лабораторная работа №2

Лабораторная работа посвящена изучению `struct`, вложенных структур, `enum`, `union`, а также работе с массивами структур и простым меню в языке C.

## Содержимое проекта

- `task1.c` — ввод и вывод данных о студенте через структуру `Student`.
- `task2.c` — массив из 5 студентов, вывод списка и поиск студента с наибольшим средним баллом.
- `task3.c` — вложенные структуры `Date` и `Person`, вывод людей и фильтрация по году рождения.
- `task4.c` — структура `Rectangle`, вычисление площади и периметра с использованием функции и указателя.
- `task5.c` — использование `enum Day` и структуры расписания занятий.
- `task6.c` — демонстрация работы `union`, `enum` и структуры с признаком типа.
- `task7.c` — мини-программа для хранения списка студентов: добавление, вывод, поиск и сортировка.
- `main.c` — пустой файл, в работе не используется.

## Сборка

Для компиляции можно использовать `gcc`.

```
gcc task1.c -o task1
gcc task2.c -o task2
gcc task3.c -o task3
gcc task4.c -o task4
gcc task5.c -o task5

```

## Краткое описание заданий

### Задание 1

Программа считывает имя студента, возраст и средний балл, сохраняет их в структуру и выводит на экран.

### Задание 2

Программа вводит данные о 5 студентах, выводит весь список и определяет студента с максимальным средним баллом.

### Задание 3

Программа использует вложенную структуру даты рождения, выводит список людей и отдельно показывает тех, кто родился после 2000 года.

### Задание 4

Программа рассчитывает площадь и периметр прямоугольника. Для площади передаётся сама структура, для периметра используется указатель на структуру.

### Задание 5

Программа демонстрирует работу перечисления `enum Day` и вывод дня недели для текущего дня и дня занятия.

### Задание 6

Программа показывает особенности `union`: после записи нового значения старое интерпретируется иначе. Также реализована структура, которая хранит значение и информацию о его типе.

### Задание 7

Программа реализует консольное меню для работы со списком студентов:

- добавление студента;
- вывод всех студентов;
- поиск по имени;
- сортировка по среднему баллу;
- выход из программы.

## Вывод

В ходе лабораторной работы были освоены основные способы описания составных типов данных в языке C и их применение в практических задачах.
