## ❓ Теоретическая часть

### 🔸 malloc

Выделяет память в куче без инициализации.

---

### 🔸 calloc

Выделяет память и заполняет её нулями.

---

### 🔸 realloc

Изменяет размер ранее выделенной памяти.

---

### 🔸 free

Освобождает память.

---

## ⚠️ Важные моменты

* Всегда проверять результат `malloc`, `calloc`, `realloc`
* Освобождать память (`free`)
* Избегать двойного освобождения
* Следить за утечками памяти

---

## 📊 Сравнение массивов

| Тип массива  | Особенности                |
| ------------ | -------------------------- |
| Статический  | фиксированный размер       |
| Динамический | изменяемый размер          |
| Стек         | быстро, но ограничено      |
| Куча         | гибко, но требует контроля |
